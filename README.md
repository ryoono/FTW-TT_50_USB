# FTW-TT_50_USB  

## 概要  
現代の消音キーボードに反抗すべく，タイプライターを叩いた時の音を楽しめるデバイスを作成する．  
タイプライター風キーボードも売ってはいるが，それを使っては味気ないので，外付けUSBデバイスとして実装する．
